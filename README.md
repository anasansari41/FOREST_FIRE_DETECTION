# Forest Fire Detection Using Image Processing and Machine Learning

An intelligent Forest Fire Detection System that uses Image Processing and Machine Learning techniques to detect forest fires from images. The system analyzes visual features such as color, texture, and smoke patterns to identify fire at an early stage, helping reduce environmental damage and enabling faster emergency response.

## 🌲 Project Overview

Forest fires cause significant ecological and economic losses every year. This project leverages image processing and machine learning to automatically classify images as **Fire** or **No Fire**, providing a fast and reliable solution for early fire detection.

## 🚀 Features

* Upload and analyze forest images
* Image preprocessing and enhancement
* Feature extraction using image processing techniques
* Machine Learning-based fire detection
* Binary classification (Fire / No Fire)
* Prediction confidence score
* Easy-to-use interface
* Fast and accurate image analysis

2. Navigate to the project directory:
cd Forest-Fire-Detection

3. Install the required dependencies:
pip install -r requirements.txt


4. Run the application:
python app.py

## 🔍 How It Works

1. Upload a forest image.
2. The image is preprocessed using OpenCV.
3. Relevant visual features are extracted.
4. The trained machine learning model analyzes the image.
5. The system predicts whether a forest fire is present.
6. The prediction result and confidence score are displayed.
